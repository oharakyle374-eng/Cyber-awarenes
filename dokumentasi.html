1. Latar Belakang & Target Lingkungan

Projek web "Waspada Digital" ini dikembangkan sebagai respon terhadap meningkatnya insiden kejahatan siber, khususnya metode Social Engineering seperti phishing dan penyebaran malware melalui berkas .APK palsu di masyarakat.

Target Lingkungan: Masyarakat umum dan pelaku UMKM lokal di wilayah Madura yang saat ini sedang aktif bertransaksi digital namun belum memiliki pemahaman teknis yang cukup mengenai cara melindungi kredensial data pribadi mereka.

2. Arsitektur dan Komponen Website

Website edukasi ini dibangun menggunakan pendekatan web statis modular dengan komponen sebagai berikut:

Nama File Fungsi / Konten
index.html Halaman Beranda, pengenalan sistem, dan peta navigasi fitur utama.
artikel.html Edukasi mendalam mengenai anatomi serangan siber dan 4 pilar tips keamanan data.
simulasi.html Zona praktik interaktif untuk mendeteksi bahaya link phishing dan chat APK palsu secara real-time.
kuis.html Evaluasi pemahaman user menggunakan sistem bank soal acak via JavaScript (Fisher-Yates Shuffle).
css/style.css Pusat kendali gaya, pewarnaan, tipografi, dan tata letak responsif website.

3. Panduan Penggunaan Website (User Guide)

  1. Pengguna membuka halaman utama (index.html) untuk memahami esensi web, lalu menekan tombol Mulai Belajar.
  2. Pengguna diarahkan ke halaman Artikel & Tips untuk menyerap materi teori keamanan dasar.
  3. Setelah membaca materi, pengguna dapat menguji insting mereka di halaman Simulasi dengan menganalisis kasus penipuan WhatsApp/Email dan menekan opsi tindakan. Sistem akan langsung memberikan feedback edukasi otomatis.
  4. Langkah terakhir, pengguna menguji pemahaman total di menu Kuis. Setiap kali halaman dimuat ulang, soal akan otomatis teracak untuk memastikan kejujuran dan variasi evaluasi.

4. Kesimpulan Produk

Melalui produk edukasi berbasis web interaktif ini, diharapkan mata rantai terlemah dalam keamanan informasi (faktor manusia) dapat diperkuat. Pengguna tidak hanya sekadar membaca teori, melainkan melatih kepekaan lewat simulasi taktik kejahatan siber yang riil terjadi di lapangan.
